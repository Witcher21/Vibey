Vibey is an intelligent AI companion built to help you answer questions, brainstorm, analyze files, and search the web. Designed with a stunning UI, dark mode aesthetics, and robust functionality, Vibey ensures a premium user experience akin to modern AI applications.

## ğŸ”¥ Features

- **Conversational Intelligence:** Chat securely with state-of-the-art AI models (DeepSeek & Groq).
- **Persistent Chat History:** Seamlessly manage multiple conversation sessions. Your chat history is saved securely and loaded instantly on return.
- **Web Search Integration:** Connect your AI directly to the internet for real-time answers.
- **Document Analysis:** Upload PDFs, Code, and Text files directly into the chat for instant summaries and data extraction.
- **User Memory Context:** The AI remembers your preferences and facts, making conversations more personalized over time.

- **Markdown & Syntax Highlighting:** Perfectly formatted AI responses with code highlighting.
